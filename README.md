# MovieFinder

This is a Movie Finder Project, Where user can search for movies and will get the desired Result.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Kodiererin/MovieFinder.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

## Tech Stack

**Client:** React, HTML , BootStrap, CSS

**Server:** Node
